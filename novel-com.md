# 番茄男频三件套标准提示词合集

这份合集按三类 skill 拆开：

- 大纲版
- 正文续写版
- 改稿审稿版

每条提示词都尽量做成可以直接复制使用的版本。你只要把中括号里的内容替换掉就能跑。

------

## 一、使用前的总原则

先记住 5 条：

### 1. 输入越具体，输出越稳

不要只写“帮我写精彩一点”。
要写清楚：

- 题材
- 卖点
- 本章目标
- 冲突点
- 想强化的方向

### 2. 一次只解决一个核心问题

不要一条提示词里既要改结构、又要写正文、又要做书名、又要审稿。

### 3. 先结构，后正文，最后改稿

推荐顺序永远是：
**大纲版 → 正文续写版 → 改稿审稿版**

### 4. 重点写“本章目标”

对正文类输出来说，本章目标是最关键的输入。

### 5. 不要把 AI 初稿直接发出去

至少过一轮改稿审稿版，再自己人工看一遍。

------

# 二、大纲版标准提示词

大纲版最适合做：

- 开书立项
- 拆前 50 章
- 重排剧情节奏
- 卡文时补后续结构

------

## 提示词 1：开书基础版

适合第一次做题材判断。

```text
请按番茄男频风格，帮我规划一本适合连载的小说。

题材：[玄幻升级 / 历史争霸 / 都市脑洞]
卖点：[一句话卖点]
目标字数：[例如 150 万字 / 200 万字]

要求：
1. 判断这个题材是否适合番茄男频
2. 提炼一句话核心卖点
3. 给出故事总纲
4. 给出适合番茄节奏的前 50 章细纲
5. 强调前 10 章的钩子、爽点和追更感
6. 风格偏连载网文，不要文艺腔
```

------

## 提示词 2：前 50 章细纲强化版

适合已经确定题材，重点要细纲。

```text
请根据以下设定，输出一份适合番茄男频的前 50 章细纲。

题材：[玄幻升级 / 历史争霸 / 都市脑洞]
卖点：[一句话卖点]
目标字数：[数字]
主角设定：[主角身份、能力、初始处境]
主线目标：[例如复仇、崛起、争天下、暴富、吞并势力]

要求：
1. 每章都要有明确推进
2. 每 3 到 5 章至少一个小高潮
3. 每章结尾尽量留下追更钩子
4. 前 10 章必须快速立住主角和卖点
5. 前 30 章内要形成稳定升级链或势力冲突链
6. 输出格式为：章名 + 本章核心事件 + 本章爽点/冲突 + 章尾钩子
```

------

## 提示词 3：三种开局方案版

适合你还没想清楚从哪种开头切入。

```text
请按番茄男频风格，为下面这个题材设计 3 种不同的开局方案。

题材：[题材]
卖点：[卖点]
目标字数：[数字]

要求：
1. 每种方案都给出开篇切入点
2. 说明前三章怎么立钩子
3. 说明前十章的主要爽点安排
4. 比较这三种方案各自适合什么风格
5. 最后推荐最适合番茄连载的一种
```

------

## 提示词 4：中期卡文补结构版

适合写到中段卡住。

```text
我这本番茄男频小说写到中期卡住了，请帮我重排后续剧情结构。

题材：[题材]
卖点：[卖点]
目标字数：[数字]
目前进度：[例如写到第 38 章 / 第 80 章]
当前问题：[例如升级线断了、反派压迫感不够、剧情开始水]
已有剧情概括：[粘贴简述]

要求：
1. 先判断问题出在哪里
2. 给出后续 10 到 20 章的推进建议
3. 强化爽点密度和冲突密度
4. 避免重复打怪或重复打脸
5. 保持番茄男频追更节奏
```

------

## 提示词 5：题材筛选对比版

适合你手里有多个方向，不知道先写哪个。

```text
我有 3 个番茄男频题材想法，请你帮我对比哪个更适合开书。

题材 1：[内容]
题材 2：[内容]
题材 3：[内容]

要求：
1. 从点击感、开局钩子、持续连载空间、爽点密度、番茄适配度几个维度分析
2. 分别给出一句话卖点
3. 说明每个题材最容易踩的坑
4. 最后选出一个最适合优先开的方向
```

# 三、正文续写版标准提示词

正文续写版最适合做：

- 日更出单章
- 续写上一章
- 按大纲落地成正文
- 给你一版可改的初稿和精修稿

------

## 提示词 1：单章续写标准版

最常用的一条。

```text
请按番茄男频风格续写一章正文。

上一章正文：
[粘贴上一章]

大纲：
[粘贴本章对应大纲]

本章目标：
[写清楚这一章必须完成什么]

要求：
1. 先保证和上一章衔接自然
2. 节奏要快，冲突明确
3. 对话简洁，避免书面腔
4. 要有明确推进，不要水文
5. 章尾留钩子
6. 输出单章正文初稿 + 去 AI 味精修版
```

------

## 提示词 2：打斗爆点强化版

适合玄幻升级、高武、战争场面。

```text
额外要求：
1. 强化战斗节奏
2. 动作描写更利落
3. 爽点要集中释放，不要拖
4. 反派压迫感要足，不要太蠢
5. 章尾要留下更大的麻烦或悬念
6. 输出初稿和去 AI 味精修版
```

```text
5. 章尾留下新的政治风险或人物冲突
```

# 四、改稿审稿版标准提示词

改稿审稿版最适合做：

- 审当前章节
- 诊断问题
- 去 AI 味
- 提高可读性
- 强化章尾
- 减少水文感

------

## 提示词 1：标准审稿版

最常用。

```text
请按番茄男频标准，对下面这章进行审稿和改稿。

章节正文：
[粘贴正文]

本章目标：
[这一章原本想达到什么效果]

要求：
1. 先指出这章最影响阅读体验的 3 到 5 个问题
2. 判断是否存在 AI 味、水文感、节奏慢、说明过多等问题
3. 给出明确修改策略
4. 输出一版精修稿
5. 保持番茄男频风格，不要文艺化
```

------

## 提示词 2：去 AI 味专项版

适合觉得句子太像模型写的。

```text
请重点帮我把下面这章去 AI 味，改得更像真人作者写的番茄男频正文。

章节正文：
[粘贴正文]

本章目标：
[粘贴]

要求：
1. 删除重复解释和空洞心理描写
2. 减少书面腔和总结腔
3. 对话更短、更自然
4. 减少“所有人都震惊了”这类围观反应
5. 保留原剧情和主要信息
6. 输出问题诊断 + 去 AI 味精修版
```

------

## 提示词 3：章尾钩子强化版

适合这一章写完太平。

```text
请按番茄男频追更逻辑，重点强化下面这一章的章尾钩子。

章节正文：
[粘贴正文]

本章目标：
[粘贴]

要求：
1. 先分析现在章尾为什么不够抓人
2. 给出 3 种不同风格的章尾钩子方案
3. 分别偏向：悬念型、危机型、反转型
4. 最后把整章顺带修到自然过渡进新章尾
5. 输出诊断 + 精修版
```

------

## 提示词 4：节奏压缩版

适合一章太散、太拖。

```text
请帮我把下面这章压紧节奏，改成更适合番茄男频追更的版本。

章节正文：
[粘贴正文]

本章目标：
[粘贴]

要求：
1. 找出水文段落和推进不足的地方
2. 删掉重复信息和无效铺垫
3. 保留关键剧情，但提高信息密度
4. 强化冲突和推进速度
5. 输出问题诊断 + 节奏压缩后的精修稿
```

------

## 提示词 5：打斗场面修正版

适合打戏看着不够爽。

# 六、最好用的“本章目标”写法

你后面真正最常用到的，反而不是整条提示词，而是“本章目标”这句话。

下面给你一些可直接套的写法。

### 玄幻升级

- 主角第一次公开越级击败对手，并引出更高层敌意
- 主角吞噬新能力，但暂时隐藏底牌
- 宗门长老开始注意到主角异常，但先不彻底翻脸
- 完成第一次真正意义上的打脸，同时埋下后续更大压迫

### 历史争霸

- 主角借一次小胜立住威望，但真正的政治风险才刚开始
- 表面上是议和，实际上是试探和布局
- 主角第一次在众人面前显露手腕，但不能完全暴露野心
- 完成一场局部胜利，并顺势挑动敌方内部矛盾

### 都市脑洞

- 主角第一次公开展示能力，并完成一轮反转打脸
- 主角表面吃亏，暗中完成布局
- 完成第一次财富/身份/能力上的跃升，并引来更高层关注
- 让之前看不起主角的人开始后悔，但先不彻底清算

# 七、你最该避免的提示词写法

这些写法很容易让输出发虚：

### 不推荐 1

“帮我写一章精彩的小说。”

### 不推荐 2

“写得爽一点。”

### 不推荐 3

“像热门小说那样写。”

### 不推荐 4

“帮我改得更好。”

### 不推荐 5

“一口气把后面全写出来。”

更好的写法永远是：

- 明确题材
- 明确卖点
- 明确本章目标
- 明确你要强化的方向

# 九、最后的建议

你后面真正常用的，不会是几十条花哨提示词，而是下面这个固定工作流：

1. 用大纲版把结构定清楚
2. 用正文续写版把本章写出来
3. 用改稿审稿版压实质量
4. 自己再手动改最后一轮

最关键的一点是：
**每次都把“本章目标”写清楚。**

这比你堆更多修饰词有用得多。
