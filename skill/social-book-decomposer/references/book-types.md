# 书型分支模板

当书籍类型较明确时，不要只用统一模板生硬输出。应先判断该书更接近哪一类，再在固定章节精读框架上叠加对应关注点。

可用的默认书型分支有四类：

1. 理论书
2. 历史书
3. 社会学 / 政治学书
4. 访谈 / 田野材料型作品

若一本书同时具备多种属性，选择一个“主分支”并可少量吸收次分支关注点，但不要混成失焦的万能模板。

## 一、理论书

### 识别信号

- 大量讨论概念、范畴、命题、理论关系、学派争论
- 作者主要在做定义、区分、批判、重构、抽象推演
- 经验材料存在，但主要用于说明理论而不是构成主体

### 精读重点

1. 问题意识：作者为什么认为旧概念、旧理论、旧框架不够用。
2. 概念工程：作者如何重新定义术语，哪些定义是全书支点。
3. 论证结构：作者如何从概念区分走向命题，再走向更大的理论框架。
4. 对手图谱：作者在回应谁、反驳谁、修正谁、继承谁。
5. 推理跳步：哪些地方理论上漂亮，但经验支撑偏弱。

### 章节精读附加要求

在 `## 论证链条拆解` 中额外回答：
- 这一章主要在做“定义 / 区分 / 批判 / 重构 / 综合”中的哪几项？
- 作者最关键的概念转折发生在哪里？
- 哪个命题是后续章节的理论支点？

在 `## 容易被忽略的细节` 中优先回收：
- 定义中的限定词
- 概念适用范围
- 与邻近概念的边界线
- 作者对反对意见的回应方式

### 全书综合附加要求

- 单独列出“核心概念网络”
- 单独列出“作者对哪些理论传统进行了修正或挑战”
- 在 `02_核心论证.md` 中标记“理论支点章节”

## 二、历史书

### 识别信号

- 以时间推进、历史阶段、事件链条、制度演变为主轴
- 大量出现年份、时期、人物、地区、档案、史料来源
- 解释重点常在“为什么在那个时间点发生变化”

### 精读重点

1. 时间线：事件先后关系与分期方式。
2. 因果链：哪些事件是起点、转折点、结果点。
3. 史料来源：作者主要依赖哪些档案、回忆录、统计、报刊、二手研究。
4. 历史解释：作者如何解释连续性、断裂、转折与路径依赖。
5. 叙述与论证的区别：不要把叙事顺序误当成充分证明。

### 章节精读附加要求

在 `## 证据、案例与材料` 中额外整理：
- 关键年份 / 时段
- 关键人物 / 机构 / 地点
- 关键史料类型
- 本章最重要的转折事件

在 `## 图像、图表与表格信息` 中特别关注：
- 地图
- 年代表
- 事件时间线
- 历史统计表

在 `## 容易被忽略的细节` 中优先回收：
- 时间顺序中的小转折
- 史料可信度限制
- 作者对史学争议的处理
- 长期结构因素与短期触发因素的区别

### 全书综合附加要求

- 生成“全书时间线摘要”
- 单独标出“关键转折点”
- 区分“背景条件 / 触发事件 / 后果扩散”

## 三、社会学 / 政治学书

### 识别信号

- 关注制度、结构、权力、行动者、机制、比较、因果解释
- 常出现案例比较、变量关系、机制分析、制度安排、群体互动
- 可能兼有规范判断，但主体仍是社会科学解释

### 精读重点

1. 研究问题：作者到底在解释什么现象。
2. 因果机制：作者认为通过什么机制产生结果。
3. 行动者与结构：谁在行动，谁受约束，制度如何塑造结果。
4. 比较框架：作者如何进行跨国家、跨群体、跨时期比较。
5. 证据与外推：案例能够说明什么，不能说明什么。

### 章节精读附加要求

在 `## 论证链条拆解` 中额外回答：
- 作者提出的是因果解释、机制解释、类型学解释，还是规范批判？
- 哪些变量或条件最关键？
- 结论的适用范围到哪里为止？

在 `## 关键概念与概念区分` 中特别关注：
- 结构 vs 行动者
- 制度 vs 策略
- 相关 vs 因果
- 机制 vs 结果
- 描述 vs 规范

在 `## 前提、限制与例外` 中优先回收：
- 比较对象的选择偏差
- 案例外推限制
- 机制成立的条件
- 被作者暂时搁置的替代解释

### 全书综合附加要求

- 单独列出“核心机制图”
- 单独列出“关键变量 / 条件表”
- 单独列出“替代解释与作者回应”

## 四、访谈 / 田野材料型作品

### 识别信号

- 大量一手材料：访谈、观察、田野笔记、口述史、案例描写
- 作者经常在“材料呈现”与“分析解释”之间切换
- 人物、场景、叙事片段对理解很重要

### 精读重点

1. 材料来源：访谈对象是谁，样本如何获得，场景在哪里。
2. 材料功能：材料是在提供厚描、举例、对照，还是直接支撑命题。
3. 方法位置：作者如何说明自己的进入方式、观察位置、偏差与限制。
4. 叙事与分析切换：哪些段落是在呈现材料，哪些段落是在上升为概念。
5. 代表性问题：个案是否被过度扩展。

### 章节精读附加要求

在 `## 证据、案例与材料` 中额外整理：
- 关键人物 / 群体
- 场景与地点
- 材料类型（访谈、观察、文本、档案）
- 引文或案例在论证中的作用

在 `## 容易被忽略的细节` 中优先回收：
- 叙事细节如何支持理论判断
- 样本边界
- 作者的位置性与方法限制
- 哪些故事只是说明性材料，哪些故事真正承担证据功能

### 全书综合附加要求

- 单独列出“材料来源与样本边界”
- 单独列出“代表性人物 / 场景索引”
- 区分“厚描段落”与“分析段落”的总体分布

## 分支选择规则

1. 先看全书主任务，而不是局部章节风格。
2. 优先选择最能决定精读重点的那个分支。
3. 若无法判断，默认使用“社会学 / 政治学书”分支，但必须在 `00_readme.md` 说明这是暂定选择。
4. 若一本书明显跨界，例如“理论性很强的历史书”，可标记为“历史书（辅以理论书关注点）”，但正文仍要有明确主次。

## 输出要求

在 `00_readme.md` 顶部新增：
- 书型判断：<理论书 / 历史书 / 社会学 / 政治学书 / 访谈 / 田野材料型作品 / 混合型>
- 判断依据：<2 到 5 句>
- 采用的附加关注点：<列出实际启用的分支规则>

在每个章节文件中，除固定结构外，可按书型在相关小节内加入小标题；不要删除固定结构。


## Mermaid 分支补充

不同书型在全书级输出中，若需要图示，优先这样画：

- 理论书：用 Mermaid mindmap 或 flowchart 画“概念网络”“理论对手与修正对象”
- 历史书：用 Mermaid timeline 画“时间线摘要”，用 flowchart 画“转折链条”
- 社会学 / 政治学书：用 Mermaid graph 或 flowchart 画“核心机制图”“变量关系图”
- 访谈 / 田野材料型作品：用 Mermaid flowchart 画“材料来源—场景—分析命题”的连接图
