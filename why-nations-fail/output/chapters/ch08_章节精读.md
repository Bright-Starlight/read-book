# 别在我们的领土：发展的障碍

## 本章在全书中的位置

**殖民地案例章（第一部分）**。本章通过鄂图曼帝国、俄罗斯、西班牙等案例，说明专制政权如何阻碍工业化。

本章与前后章节的关系：
- 第7章（光荣革命）→本章（专制政权阻碍发展）→第9章（英国殖民地案例）
- 本章对比第7章——为什么有些国家能发展，有些不能

## 本章要回答的核心问题

**专制政权和缺乏政治集权的国家如何阻碍了工业化和经济发展？西班牙、俄罗斯、奥斯曼帝国等为什么没能建立广纳式制度？**

## 本章的核心主张

### 核心命题一：专制政权恐惧创意破坏

**印刷机的案例**：
- 西欧：1470年代快速传播
- 鄂图曼：1485年禁止，1727年才允许

**为什么专制政权禁止印刷**：
- 印刷术可能传播新思想
- 威胁专制统治的意识形态基础
- 精英恐惧创意破坏

### 核心命题二：西班牙科尔蒂斯的失败

**英格兰国会 vs 西班牙科尔蒂斯**：

| | 英格兰国会 | 西班牙科尔蒂斯 |
|---|---|---|
| 代表 | 城市+农村 | 只有城市 |
| 权力 | 征税+立法 | 有限 |
| 结果 | 限制王权 | 被国王边缘化 |

**为什么失败**：
- 代表范围太窄（只有18个城市）
- 未能形成多元利益联盟
- 国王容易各个击破

### 核心命题三：俄罗斯的农奴制

**俄罗斯农奴制的特殊性**：
- 农奴被束缚在土地上
- 不能自由迁移
- 农业生产力极低

**为什么持续**：
- 地主贵族需要农奴劳动
- 专制政权依赖贵族支持
- 压制农民创新激励

## 论证链条拆解

### 步骤1：奥斯曼帝国的案例

**奥斯曼帝国的制度特征**：
- 专制苏丹制
- 土地国有，包税制度
- 压制市场和技术创新

**为什么阻碍发展**：
- 印刷机被禁止（1485-1727）
- 创意破坏威胁专制
- 缺乏产权保护

### 步骤2：西班牙科尔蒂斯的失败

**西班牙议会的演变**：
- 科尔蒂斯：中世纪议会
- 代表只有大城市贵族
- 被国王边缘化

**英格兰国会的成功**：
- 代表城市+农村+新贵族
- 形成限制王权的多元联盟
- 权利法案最终确立国会至上

### 步骤3：俄罗斯农奴制

**农奴制的经济影响**：
- 农民被束缚→没有劳动力流动
- 技术停滞→农奴没有改进技术的激励
- 地主不投资→农业技术落后

### 步骤4：缺乏政治集权的问题

**阿富汗、海地、索马里**：
- 没有政治集权
- 无法建立基本秩序
- 工业化无法起步

**关键洞见**：**光有政治集权不够（变成专制），光有政治多元化也不够（无政府）。需要两者结合。**

## 证据、案例与材料

### 证据1：印刷机的传播与禁止

- **类型**：比较历史案例
- **功能**：说明专制政权如何系统性地压制创新
- **对比**：西欧1470年代 vs 奥斯曼1727年才允许
- **强度**：高

### 证据2：西班牙科尔蒂斯 vs 英格兰国会

- **类型**：比较政治案例
- **功能**：说明为什么有些议会成功了，有些失败了
- **关键**：代表范围和多元利益联盟
- **强度**：高

### 证据3：俄罗斯农奴制

- **类型**：历史案例
- **功能**：说明榨取式制度如何压制创新激励
- **机制**：农奴被束缚→不投资技术
- **强度**：高

## 前提、限制与例外

### 作者隐含的前提

1. **技术进步需要制度环境**：同样的技术在不同制度下有不同结果
2. **精英恐惧创意破坏是结构性的**：不是个人选择，而是制度激励

### 适用范围

- 本章案例主要适用于**欧洲和奥斯曼帝国**
- 对亚洲、非洲的适用性需要具体分析

## 容易被忽略的细节

### 细节1：印刷机不只是技术——奥斯曼帝国的案例

印刷机的传播意味着：
- 知识普及
- 新思想的扩散
- 对专制意识形态的挑战

**奥斯曼帝国印刷机禁令的具体历史**：

| 时间 | 事件 |
|------|------|
| 1450年代 | 谷登堡在德国发明活字印刷 |
| 1475 | 意大利出现第一批印刷书籍 |
| 1485 | 苏丹巴耶塞特二世禁止印刷书籍（阿拉伯文） |
| 1493 | 西班牙宗教裁判所成立 |
| 1502 | 苏丹塞利姆一世继续禁令 |
| 1727 | 第一部土耳其语印刷书在伊斯坦布尔出版（苏丹艾哈迈德三世允许） |

**为什么奥斯曼帝国要等242年才允许印刷**：
- 印刷机可能被用来传播异端思想
- 奥斯曼帝国的统治合法性基于伊斯兰教法（Sharia）
- 新的宗教解释可能威胁现状
- 毛拉（伊斯兰学者）阶层担心印刷会让他们失业
- 统治者担心知识普及会动摇权威

**对比欧洲**：
- 1470年代：印刷机快速传播到整个西欧
- 1517年：马丁·路德的《九十五条论纲》通过印刷迅速传播——这在奥斯曼帝国是不可想象的
- 结果：宗教改革、启蒙运动、科学革命都在西欧发生

**1727年后为什么允许**：
- 艾哈迈德三世时期，苏丹权力相对弱化，需要商人阶层支持
- 军事技术和行政管理的需要迫使苏丹接受印刷
- 奥斯曼帝国与欧洲的竞争压力——没有印刷，英帝国无法现代化

### 细节2：西班牙科尔蒂斯的代表性问题——为何失败

科尔蒂斯（Cortes）只代表城市贵族：

**西班牙科尔蒂斯的历史演变**：

| 时期 | 代表性 | 权力 |
|------|--------|------|
| 12-13世纪 | 莱昂、卡斯蒂利亚的科尔蒂斯有广泛代表（贵族、教会、城市） | 相对强大，可拒绝国王的税收要求 |
| 14-15世纪 | 代表范围缩小，贵族影响力增强 | 逐渐被边缘化 |
| 1469 | 费迪南德（阿拉贡）与伊莎贝拉（卡斯蒂利亚）联姻 | 统一西班牙的开始 |
| 1480年代 | 科尔蒂斯被王室控制 | 成为橡皮图章 |

**英格兰国会的对比**：

| 特征 | 英格兰国会 | 西班牙科尔蒂斯 |
|------|-----------|--------------|
| 代表基础 | 郡县（农村）+城市 | 主要是城市 |
| 农村代表 | 骑士代表郡县 | 没有农村代表 |
| 贵族 | 上议院（贵族） | 直接参与科尔蒂斯 |
| 新兴阶层 | 商人和专业阶层 | 几乎没有代表 |
| 利益多元性 | 高（多地区、多阶层） | 低（集中在旧贵族） |

**为什么西班牙科尔蒂斯失败了**：
1. **代表范围太窄**：只有大城市贵族，没有新兴商人阶层
2. **贵族利益一致**：西班牙贵族与王室联姻，利益趋于一致
3. **美洲殖民的红利**：西班牙王室从美洲获得大量金银，不需要向科尔蒂斯要钱
4. **政治整合失败**：西班牙实际上是多个王国的联合，而非统一的民族国家

**关键结果**：
- 1492年收复失地运动完成后，伊莎贝拉和费迪南德已经建立了绝对王权
- 科尔蒂斯被边缘化，王室可以自行决定税收和政策
- 这就是为什么西班牙最终走向榨取式制度

### 细节3：政治集权vs专制——俄罗斯农奴制的案例

政治集权和专制不是一回事：
- 好的政治集权：建立秩序，提供公共品
- 坏的专制：利用权力为少数精英服务

关键是**是否有政治多元化制衡**。

**俄罗斯农奴制的具体历史**：

| 时期 | 关键事件 | 对农奴制的影响 |
|------|---------|--------------|
| 1497 | 《尤里耶夫法令》 | 首次限制农民在圣尤里节迁移 |
| 1581 | 沙皇费奥多尔一世 | 颁布"禁止出走令"，禁止农民逃离地主 |
| 1592 | 土地清丈 | 正式将农民登记在土地上 |
| 1649 | 《会议法典》 | 完全取消农民迁移权，农奴制合法化 |
| 1700s | 彼得大帝 | 进一步强化对农民的控制 |
| 1861 | 亚历山大二世改革 | 正式废除农奴制 |

**为什么农奴制如此根深蒂固**：
- 俄罗斯贵族（Dvoryan）几乎完全依赖农奴劳动
- 沙皇政权与贵族精英紧密联盟——农奴制是这种联盟的基础
- 俄罗斯地广人稀，没有英格兰那样的劳动力市场竞争压力
- 东正教支持专制体制——为农奴制提供意识形态合法性

**为什么英格兰走向了不同的道路**：
- 黑死病后劳动力短缺，农民可以用脚投票
- 议会制度使国王不能单方面强化农奴制
- 商人和新贵族的利益与农奴制相反——他们需要自由劳动力
- 英格兰没有"绝对王权"的政治传统

**俄罗斯农奴制的经济后果**：
- 1861年废除时，农业生产力仍停留在1700年代水平
- 俄国工业化因此滞后——工厂需要自由劳动力
- 直到1860年代铁路建设后，俄国才真正开始工业化

**关键洞见**：俄罗斯的问题是**政治集权服务于少数贵族**，而非建立普遍秩序。这与英格兰光荣革命后的情况形成鲜明对比。

## 一分钟回看

**本章核心洞见**：专制政权系统性地阻碍工业化和创意破坏。奥斯曼帝国禁止印刷机172年；西班牙科尔蒂斯被国王边缘化；俄罗斯农奴制压制农民创新激励。这些案例与英国光荣革命形成鲜明对比——为什么有些国家能建立广纳式制度，有些不能？关键在于：是否形成了足够广泛的政治联盟来限制专制权力；是否有政治多元化来制衡集权。

**值得回看**：本章与第7章形成对比。第7章说光荣革命建立了广纳式制度；本章说专制政权阻碍了广纳式制度的建立。理解这两个过程，才能理解为什么有些国家富裕，有些国家贫困。
