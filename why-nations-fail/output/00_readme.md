## 核心命题

**国家贫富的根源在于制度：榨取式制度（extractive institutions）创造贫困，广纳式制度（inclusive institutions）创造富裕。**

- 广纳式政治制度：权力多元、广泛分配政治权力，建立约束执政者的多元政治制度
- 广纳式经济制度：保护私有财产、开放竞争、支持创意破坏
- 榨取式制度：由少数精英控制政治权力，建立为他们服务的经济制度
