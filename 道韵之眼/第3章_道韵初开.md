# 第3章 道韵初开

后山。

枯井旁。

林尘靠着井沿坐下，大口大口地喘息着。

月光从头顶洒落，四周只有虫鸣。枯井早已干涸多年，井壁上爬满了青苔，在月色下泛着幽幽的绿光。

这里是青云门后山的偏僻角落，平日里根本不会有人来。

林尘也不知道自己跑了多久。他只知道，离开那片废墟之后，他一路狂奔，穿过杂役房的后巷，绕过演武场后面的竹林，最后来到了这个他曾经无数次偷偷来过的枯井旁。

十年前，师父第一次带他来过后山，指着这口枯井说："徒儿，以后若是修炼上有什么困惑，就来这里打坐。"

那时候他不懂为什么。现在他懂了——这里是整个青云门灵气最浓郁的地方之一，只是被某种阵法遮蔽了，寻常人根本感知不到。

他坐在井沿上，脑子里一片混乱。

师父死了。

死在紫霄阁的围杀之中。

他的师父，那个总是拍着他的肩膀说"不要放弃"的老人，那个在山下救了他一命的恩人，就这样没了。

林尘攥紧拳头，指甲几乎嵌进肉里。

他想起师父临终前的遗言——

"不要相信任何人，去丹霞宗。"

丹霞宗到底是什么地方？师父为什么要让他去哪里？那里有什么在等着他？

还有那个神秘的天道盟，与紫霄阁又是什么关系？

师父说他的眼睛是"道韵之眼"，能看穿一切，是被某些势力觊觎的东西……

那些势力，会不会就是追杀师父的人？

无数个问题在脑海中盘旋，却没有答案。

林尘深吸一口气，将那些纷乱的思绪压下。

现在不是悲伤的时候。

他还活着，就必须活下去。必须变强。必须找到师父死亡的真相。

而要做到这些，首先他得活着。

他摸了摸怀里的绢帛。

《道韵心经》。

师父临死前交给他的东西。

林尘颤抖着将绢帛取出，在月光下缓缓展开。

泛黄的绢帛上，密密麻麻的小字清晰可见。那是一种他从未见过的古文字，笔画像是用利器刻上去的，透着一股沧桑而神秘的气息。

第一句便是：

"道韵者，天地之理，万物之序。悟其韵者，可破天人之障。"

林尘的心跳加速。

他继续往下看。

"道韵心经分九重：第一重名曰'感知'，可令修炼者感知天地灵气之流向，辨万物之性；第二重名曰'洞彻'，可令修炼者看穿万物之本相，察破绽之所在；第三重名曰'补全'，可令修炼者修补残缺功法之漏，融会贯通；第四重名曰'预判'，可令修炼者料敌先机，洞悉敌意……"

后面的字迹模糊不清，似乎是年代太过久远，墨迹已经褪色。

但前面几重的内容，已经足够让林尘心神震动。

这卷功法……和他的眼睛能看到的东西，完全吻合！

今天测试的时候，他能看见经脉中灵气的流动，能看见堵塞的真正原因——这不就是"感知"境界吗？

而他能看穿苏晴雪身上紫色道韵的异常，这不就是"洞彻"境界吗？

只是他自己都不知道，原来这就是道韵心经描述的能力！

"道韵之眼……"林尘喃喃自语，"原来师父给我的，是这个东西……"

他的心中涌起一股复杂的情绪。

师父早就知道他的眼睛不正常吗？为什么要把这卷功法交给他，却不教他修炼方法？

是在等他自己觉醒吗？

还是说……师父有什么难言之隐？

林尘来不及多想。

不管怎样，现在他必须变强。

他照着第一层心法，开始尝试修炼。

盘膝而坐。双手结印。

呼吸。吐纳。引灵气入体。

一开始，什么都没有发生。

林尘有些着急。他闭上眼，努力感知周围的灵气。

但就在这时——

他感觉到了。

一丝若有若无的灵气，从四面八方涌来。

那是一种很奇妙的感觉。灵气像看不见的水流，从天地间汇聚而来，在他的身周形成一个看不见的漩涡。

它们试探性地触碰着他的经脉，像是在确认什么。

然后——

"轰！"

某一刻，那些灵气像找到了入口一般，疯狂地涌入林尘的经脉！

温热的力量在丹田里汇聚，比白天在测试石前感知到的还要清晰、还要浓郁十倍！

这就是灵气入体的感觉！

这就是修炼的感觉！

林尘的身体不由自主地震动了一下。

他感受到了经脉中奔涌的力量，感受到了丹田在一点一点地被填满。那种感觉太美妙了，像是久旱逢甘霖，像是暗夜中看到了光明。

一个时辰过去了。

两个时辰过去了。

月亮从东边移到了天顶，又从天顶慢慢往西边落去。

林尘浑然不觉，完全沉浸在修炼的状态中。

他能感觉到自己的丹田在一点一点地被填满，那股温热的力量越来越强，在经脉中奔涌，冲击着那道看不见的屏障。

那屏障，是十年来堵在他经脉中的杂质。

现在，在道韵心经的引导下，那些杂质正在被一点一点地溶解。

"轰隆隆……"

林尘的脑海中响起一阵轰鸣。

十年的积累，在这一刻终于找到了出口！

他感觉到了——那股力量在丹田里凝聚成形，化作一颗米粒大小的灵核，静静地悬浮在丹田正中央。

练气一层。

他终于突破了！

"这就是……灵气入体的感觉……"

林尘缓缓睁开眼睛。

他的瞳孔中，有淡淡的青光一闪而过。

他握紧拳头，感受着经脉中奔涌的力量。那股力量比之前强了十倍不止，让他有一种脱胎换骨的感觉。

他的感知也比之前敏锐了十倍。他能感觉到周围的灵气在流动，能听见远处树叶落地的细微声响，甚至能隐约感知到青云门主峰那边残留的灵力波动。

主峰方向，还有微弱的战斗声传来。

看来，紫霄阁的袭击还没有完全结束。

林尘站起身来，心中涌起一股复杂的情绪。

他变强了。虽然只是练气一层，但这只是开始。有了道韵心经，他相信自己一定能变得更强。

但师父已经死了。

他想起师父临终前的遗言——"去丹霞宗"。

丹霞宗到底是什么地方？

师父为什么让他去哪里？

林尘攥紧拳头。

不管怎样，他必须活下去。必须变强。必须找到师父死亡的真相。

而要做到这些，首先他得回到门派里，不被人发现他偷偷跑到后山来修炼。

而且，他必须弄清楚门派里还有没有敌人。

紫霄阁为什么会突然围攻青云门？师父为什么会被杀？

这些问题不解决，他永远无法安心。

林尘正准备起身离开——

"林尘！"

一个阴恻恻的声音从树丛后传来。

林尘猛然回头，只见几个身影从树丛后走出。

为首的是一个中年男子，身材矮胖，穿着外门执事的服饰，脸上带着皮笑肉不笑的表情。他身后跟着两个杂役弟子，正是不久前在杂役房里嘲笑林尘的那几个。

赵德。外门执事。练气七层修为。

"好大的胆子，"赵德慢悠悠地走过来，目光在林尘身上扫了一圈，"分配你在杂役房干活，你却半夜乱跑，擅闯后山禁地！"

林尘脸色微变。

赵德是外门执事，练气七层的实力，捏死他像捏死蚂蚁一样简单。而且此人素来与师父云虚子不和，当年师父还在的时候，他就多次刁难林尘，只是碍于师父的面子不敢太过分。

现在师父死了……

"执事大人，"林尘强压下心头的慌乱，"我只是睡不着出来透透气——"

"透气？"赵德冷笑一声，"后山是门派禁地，你一个杂役也配来这里透气？"

他走近几步，目光在林尘身上打量着，忽然眼神一凝。

"等等……你身上的气息……"

赵德的脸色微变。

他是练气七层的修士，能清楚地感知到林尘身上的灵力波动——那是练气一层才有的气息，虽然微弱，却实实在在。

"你突破了？！"赵德失声惊呼，随即脸色阴沉下来，"不对……你白天的测试，明明是经脉堵塞率九十七……怎么可能一夜之间就突破练气一层？"

林尘心里一沉。

糟糕。

他刚突破，还不懂得收敛气息，被赵德察觉了。

"执事大人明鉴，"林尘快速思索着说辞，"弟子也不知道是怎么回事……今天白天测试结束之后，回到杂役房就感觉身体有些异样，像是有什么东西在体内涌动……弟子以为是生病了，没想到睡到半夜，忽然感觉丹田一阵剧痛，然后就……就突破了……"

"突破了？"赵德冷笑，"你以为我会信？一夜之间从经脉堵塞率九十七变成练气一层？这种鬼话，你留着去骗三岁小孩吧！"

他猛然抬手，一道灵力朝林尘抓去。

林尘想躲，但他刚突破练气一层，哪里是练气七层赵德的对手？

练气一层对练气七层，差距太大了。

三招之内，他就被赵德的灵力压制住，动弹不得。

"说！"赵德恶狠狠地盯着他，"你到底用了什么邪术！是不是云虚子那个老东西临死前给了你什么？"

听到"云虚子"三个字，林尘的眼睛瞬间红了。

"你不许说我师父！"

"哟？"赵德挑了挑眉，"怎么，说你师父你不乐意了？那个老东西早就该死了——"

"你闭嘴！"

林尘不知哪来的力气，竟然挣脱了赵德的灵力压制，一拳朝他挥去。

但练气一层对练气七层，差距太大了。

赵德轻描淡写地一挥手，就将林尘的攻击挡下，顺势一掌拍在林尘胸口。

"砰！"

林尘倒飞出去，重重摔在地上，胸口一阵剧痛，差点没喘上气来。

"不知死活的东西，"赵德走过来，一脚踩在林尘的手上，"既然你不说，那我就自己找。云虚子那个老东西临死前肯定给你留了东西——"

他的手按在林尘丹田位置，灵力开始侵入。

剧痛传来。

林尘感觉自己的丹田像是要被撕裂一样——那是废丹田的手法！赵德要废掉他的修为！

"不——！"

林尘拼命挣扎，但根本挣脱不了。

赵德的脸上露出狰狞的笑容："你不是很能忍吗？十年了都不叫一声苦，今天就叫出来让我听听——"

"住手！"

一道清冷的女声从夜色中传来。

赵德的手一僵。

他猛然抬头，只见月光下，一道白色身影踏步而来。

青丝如瀑，面若寒霜，眉心那点朱砂在月色下格外醒目。

她穿着一身素白长裙，周身气息冷冽如霜，仿佛不食人间烟火的仙子。

苏晴雪。

"苏……苏师姐？！"赵德脸色大变，连忙松开林尘，起身行礼，"您怎么来了这种地方？"

苏晴雪没有理他，目光落在林尘身上。

准确地说，是落在林尘腰间那枚破损的玉佩上。

她的目光微微一凝。

"这枚玉佩，"苏晴雪的声音微微一顿，"你从哪里得来的？"

林尘浑身是土，狼狈地爬起来。

他看着苏晴雪，忽然想起白天她身上那道让他完全无法理解的紫色道韵。

这一刻，他心中警铃大作。

师父临终前说"不要相信任何人"——可眼前这个女人，为什么会对师父的玉佩有反应？

这枚玉佩是丹霞宗的信物。

而苏晴雪，是内门第一天骄。

她为什么会认识？

"我……"林尘张了张嘴。

赵德却抢先一步，讨好地笑道："苏师姐，这小子就是个废物，在杂役房干了十年都没突破，今天还在测试时作弊，扰乱门内秩序，我正准备按门规处置——"

"废物？"

苏晴雪忽然弯下腰，修长的手指捏住林尘的下巴，将他的脸抬起来。

两人的目光对上。

林尘看见苏晴雪的眼睛清澈如水，深不见底。那里面没有嘲讽，没有轻蔑，只有某种让他捉摸不透的审视。

"白天测试的时候，你的经脉明明是堵塞的。"苏晴雪的声音很轻，只有他能听见，"可你现在已经是练气一层了。十天前的外门考核，你还没突破。"

林尘瞳孔骤缩。

她怎么知道？她是内门弟子，怎么会关注一个外门杂役的考核？

"这玉佩，"苏晴雪的目光锐利如剑，"到底是从哪里来的？"

林尘脑中电光火石。

他不能说实话。师父说不要相信任何人——这个女人突然出现在后山，还对这枚玉佩如此在意……

她到底是什么人？

她和杀死师父的势力有关系吗？

"路……路上捡的。"林尘移开目光，声音发虚。

苏晴雪盯着他看了三秒。

那三秒像是三年一样漫长。

林尘感觉自己的秘密仿佛被看穿了一样，浑身不自在。

然后她松开手，站起身。

"赵执事，"她的声音恢复了清冷，"这个人，我带走。"

赵德脸色难看："苏师姐，这小子犯了门规——"

"我说，"苏晴雪侧过头，目光冷冷地扫过赵德，"我带走。"

那一瞬间，赵德仿佛被什么东西扼住了喉咙，脸色涨红，却一个字也说不出来。

苏晴雪是内门第一天骄，筑基后期的修为，整个青云门年轻一代的领军人物。别说他一个外门执事，就是内门长老见了她，也要客客气气的。

而且苏晴雪身后，还站着一个强大的苏家。苏家是苍梧山脉的世家大族，势力庞大，不是他一个小小的外门执事能得罪得起的。

赵德的嘴角抽搐了一下，最终低下了头："……是。"

苏晴雪没有再看林尘，转身往后山小径走去。

"跟上。"

林尘愣了一瞬。

他不知道等待自己的是什么。但此刻，他没有选择——赵德想废他修为，而苏晴雪不知是敌是友，但至少目前没有要害他的意思。

而且，他需要知道，苏晴雪为什么会对师父的玉佩如此在意。

他咬牙爬起来，跟了上去。

赵德站在原地，脸色阴沉地望着他们消失的方向，攥紧了拳头。

"苏晴雪……"他的声音从牙缝里挤出来，"别以为你是内门第一就了不起……云虚子的徒弟落入你手里，迟早要你好看……"

他转身，带着两个杂役弟子往山下走去。

夜风呼啸。

两道身影消失在月色中。

林尘跟在苏晴雪身后，穿过后山的小径，往内门的方向走去。

他不知道苏晴雪要带他去哪里，也不知道等待自己的是什么。

但有一件事他很清楚——

这个女人，知道很多事情。

关于师父的玉佩，关于丹霞宗，关于他眼睛的秘密……

她到底是谁？

而他自己，又该何去何从？

月光洒落在小径上，将两人的身影拉得很长很长。

林尘不知道的是，就在他跟着苏晴雪离开的时候，远处的山巅上，一双眼睛正若有所思地望着这边。

"苏晴雪……她为什么要带走林尘？"

一个黑衣人站在山巅上，看着两道消失的身影，眼中闪过一丝阴沉。

"有意思……这件事，必须禀报上去……"

他的身影一闪，消失在夜色中。
