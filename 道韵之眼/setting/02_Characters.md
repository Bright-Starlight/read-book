## 林尘（主角）

| 项目 | 内容 |
|------|------|
| 身份 | 原青云门杂役弟子，现为逃亡者 |
| 境界 | 练气一层（新突破） |
| 年龄 | 17岁 |
| 外貌 | 瘦削，营养不良的蜡黄，灰色外袍 |
| 核心特质 | 表面沉默寡言，内心傲骨深藏；视觉特殊，能看穿万物"道韵流向" |
| 关键物品 | 半卷《道韵心经》、云虚子破损玉佩（丹霞宗信物）、续命草（玉佩空间内） |
| 近期状态 | 师父被杀，被苏晴雪带走，命运未知 |
| 核心秘密 | 道韵之眼继承者，被天道盟和紫霄阁追杀；不知苏晴雪真实目的 |

## 苏晴雪

| 项目 | 内容 |
|------|------|
| 身份 | 青云门内门第一天骄 |
| 境界 | 筑基后期 |
| 年龄 | 约20岁 |
| 外貌 | 身段修长，肌肤胜雪，眉心一点朱砂，素白长裙 |
| 关键特征 | 身上有"天罚之印"（紫色道韵），修炼速度奇快 |
| 关键物品 | 内门信物（曾给林尘进入藏书阁资格） |
| 近期状态 | 带走林尘，目的不明 |
| 秘密 | 天罚之印会在某天爆发使她魔化；她知道玉佩是丹霞宗信物；并非魔修后裔而是天罚之印受害者 |
| 与林尘关系 | 幼时曾有一面之缘但早已不记得 |
