## 第1章 废物

**字数**：5000+字符

**摘要**：青云门外门年度测试，林尘被鉴定为"经脉堵塞率97%"，当众羞辱。但混乱中他发现自己的眼睛能看到真相——堵塞不是真的堵塞，而是七道被压制的灵脉在以特殊方式吸收灵气，速度比常人快十倍。测试石异动，青光冲天。全场震惊。林尘发现自己是天生的修炼天才，只是体质特殊被误判十年。苏晴雪路过，她身上那道紫色道韵让林尘完全无法理解，深感不安。

### 已发生事件

- [x] 林尘被当众宣布为"废物"，经脉堵塞率97%
- [x] 林尘眼睛异变，看穿测试石真相
- [x] 发现林尘不是废物，是逆吸收体质
- [x] 测试石青光冲天，全场震惊
- [x] 苏晴雪出现，林尘注意到她身上异常紫色道韵
- [x] 林尘知道自己不是废物了

### 未解决

- 为什么十年才觉醒？（今日测试才触发）
- 苏晴雪身上的紫色道韵究竟是什么？
- 苏晴雪为什么会出现在外门演武场？
