## 第1章 废物

**字数**：5000+字符

**摘要**：青云门外门年度测试，林尘被鉴定为"经脉堵塞率97%"，当众羞辱。但混乱中他发现自己的眼睛能看到真相——堵塞不是真的堵塞，而是七道被压制的灵脉在以特殊方式吸收灵气，速度比常人快十倍。测试石异动，青光冲天。全场震惊。林尘发现自己是天生的修炼天才，只是体质特殊被误判十年。苏晴雪路过，她身上那道紫色道韵让林尘完全无法理解，深感不安。

### 已发生事件

- [x] 林尘被当众宣布为"废物"，经脉堵塞率97%
- [x] 林尘眼睛异变，看穿测试石真相
- [x] 发现林尘不是废物，是逆吸收体质
- [x] 测试石青光冲天，全场震惊
- [x] 苏晴雪出现，林尘注意到她身上异常紫色道韵
- [x] 林尘知道自己不是废物了

### 未解决

- 为什么十年才觉醒？（今日测试才触发）
- 苏晴雪身上的紫色道韵究竟是什么？
- 苏晴雪为什么会出现在外门演武场？

**摘要**：苏晴雪将林尘带到后山竹林，问玉佩来历。林尘撒谎说是在路上捡的，苏晴雪冷笑反驳——这是丹霞宗开派祖师的随身信物，失落近千年，岂是能在路边捡到的？林尘被迫承认是师父遗物，苏晴雪听到"云虚子"三个字时表情复杂，喃喃"原来是他"。苏晴雪警告林尘"离赵德远一点"后离开。林尘独自在竹林中发现玉佩内藏储物空间，内有一株金色续命草——正是能解除道韵之眼透支反噬的灵草。正在此时，外门大师兄赵昊带人出现，嘲讽林尘"勾搭苏师姐"，出手重伤他，并约他子时去后山枯井"看一样东西"。林尘明知可能是陷阱，仍决定赴约。同时，赵德在执事房召赵昊密谋，暗示今晚有大事要发生。
